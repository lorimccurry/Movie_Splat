Movie splaT
==============
###Ruby on Rails capstone project for Nashville Software School Cohort 3.

## Elevator Pitch
A Ruby on Rails app for any cinephile looking to keep track of which movies they've seen, want to see, and own. User can rate movies on their list and see how their movie tastes suck or don't suck compared to Rotten Tomatoes ratings.

## Feature List:

   * Build your own movie tracker library: enter movies
     by category: seen, want to see, own, want to own, rating (1-100 scale), comments
   * View tracker library and sort/search data by Title, Viewed, Own, etc.
   * User can see how well they rate compared to Rotten Tomatoes critics and determine if it's time to quit their day job and move to Hollywood
   * Rotten Tomatoes API data featured from OMDB

## Data Sources:

   * OMDB API
   * cinephile personal collection and viewing history
